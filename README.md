<h1>Lib_getLinksFromMd</h1>


<h2> Descrição</h2>

<p>Esta biblioteca foi desenvolvida por Sabrina Andrade para receber uma string (em formato Markdown) e extraia todos os links encontrados, devolvendo um vetor de objetos.
</p>



<h2> Instruções de Uso</h2>

<p>Para utiliza-lá, acesse seu o npm pelo seu computador e faça a instalação pelo seguinte comando: npm i lib_getlinksfrommd </p>
